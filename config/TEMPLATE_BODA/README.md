# Plantilla Base de Configuración

Esta carpeta representa la estructura estándar para una nueva boda.

Cada nuevo evento deberá copiar esta estructura.

Ejemplo:

- B001_Jimena_Jorge
- B002_Dania_Carlos
- B003_Ana_Luis

Elementos configurables:

- Nombre de los novios
- Fecha del evento
- Ciudad
- Google Sheet asociado
- URL Panel WhatsApp
- URL Confirmaciones
- Dashboard asociado
