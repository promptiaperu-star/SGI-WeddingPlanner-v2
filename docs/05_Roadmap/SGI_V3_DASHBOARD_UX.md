# SGI Wedding Planner v3

## Objetivo

Evolucionar el Dashboard actual hacia un Sistema de Gestión Multi-Boda orientado a Wedding Planner.

## Sprint UX/UI v3

### Panel Principal

* Dashboard Multi-Boda.
* Total de bodas activas.
* Tarjetas de bodas vigentes.
* Estados del proceso.
* Alertas automáticas.
* Acceso a Dashboard individual.

### Dashboard Individual

* Información del evento.
* KPIs simplificados.
* Gráfico de barras.
* Evolución diaria de confirmaciones.
* Contador regresivo.
* Alertas de pendientes.

### Flujo de Trabajo SGI

1. Lista de invitados.
2. Envío WhatsApp.
3. Reporte de confirmaciones.
4. Distribución de mesas.

## Filosofía de diseño

* Estilo minimalista.
* Paleta beige / blanco / grises.
* Bento Grid.
* Espacios amplios.
* UX centrado en Wedding Planner.
